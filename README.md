# Healthcare App

A full-stack MERN healthcare application with AI-powered features.

## Features
- AI Symptom Checker (Groq LLM)
- Find Doctors & Book Appointments
- Real-time Chat (Socket.io)
- Digital Prescriptions (PDF download)
- Health History Timeline
- Medicine Reminders (Browser Notifications)
- AI Health Report Generator

## Tech Stack
- **Frontend:** React, Socket.io-client
- **Backend:** Node.js, Express, MongoDB, Socket.io, Groq API, pdfkit
